# Simple-Youtube-to-MP3
A Python GUI app to download YouTube videos as MP3s.

## Features
- Paste YouTube URL
- Shows progress bar
- Auto-saves to Downloads folder
- Auto-opens folder after download

## Built With
- Python
- PyQt5
- yt-dlp
- ffmpeg

## How to Run
Double-click the `.exe` in `dist/` or run `vieyt2mp3.py` with Python.
